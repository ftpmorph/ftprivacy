# In its default config, Windows sends out a lot of "wpad.blahblahblah" requests.
# WPAD is an outdated Microsoft protocol from the 90's that creates a big security risk.
# https://en.wikipedia.org/wiki/Web_Proxy_Auto-Discovery_Protocol#Security
# https://www.networkworld.com/article/3105994/disable-wpad-now-or-have-your-accounts-and-private-data-compromised.html

# Some routers even block all WPAD traffic out the box, e.g. Google WiFi/Google Nest:
# "Note: To keep your network secure, Google Wifi and Google Nest Wifi do not support Web Proxy Auto-Discovery (WPAD) Protocol. WPAD can easily be hacked and expose your browsing behavior or data. If your device has WPAD enabled, we recommend you refer to your device user manual and disable it."
# https://support.google.com/wifi/answer/6246630

# WPAD is only enabled by default in Windows. If you use Windows, seriously, disable WPAD.

# This regex should block all WPAD requests for you in the meantime:

^wpad($|\.)
||wpad.*^
